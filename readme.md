## 这是什么？

LeetCode 解题本，基于**算法与数据结构**，对做过的 LeetCode 练习题进行归纳与总结。

项目内题解主要由 Golang 书写。

项目使用 [docsify](https://docsify.js.org/#/) 构建，搭建教程见 [《docsify 入坑指南与我放弃 Gitbook 的那些理由》](http://jalan.space/2019/06/21/2019/begin-docsify/)
